    My journey into web development began with the IT – Tech for Women course at Sundsgården Folkhögskola. This comprehensive program provided me with a solid foundation in front-end development, equipping me with essential skills and hands-on experience. I learned to structure web content effectively using HTML5, ensuring semantic accuracy and accessibility. With CSS3, I mastered styling web pages, focusing on responsive design techniques to ensure compatibility across various devices. JavaScript played a crucial role in my education, allowing me to develop interactive web elements that enhance user engagement and functionality. Additionally, I utilized Figma for designing user interfaces and creating prototypes, which facilitated a seamless transition from design to development. The course also introduced me to modern design libraries, such as Bootstrap, expediting the development process and ensuring consistent design. Exploring popular front-end frameworks provided insights into building robust web applications. Learning Git and GitHub enabled efficient collaboration and code management, essential for any development team. Visual Studio Code became my primary development environment, where I leveraged its powerful features and extensions to enhance productivity. The program emphasized programming techniques, teaching best practices in coding, debugging, and optimizing web applications. Working in an agile environment allowed me to collaborate with peers, manage projects through iterative development cycles, and experience the dynamics of team-based development. This education not only provided me with the technical skills necessary for front-end development but also fostered a collaborative and inclusive learning environment. My experience in this course has been instrumental in shaping my passion for web development and preparing me for more advanced challenges in the field.
